Introduction

Disclaimer: I made this as a final project for my compiler class. It is not entirely reliable or robust per se to create games with this language.

Orcinus is a language to create and play maze based games, aimed to teach people how to code. The syntax is mostly inspired by Ruby and Python, and built on top of JavaScript.


Objects

The building block of Orcinus-generated games is object. Object could range from game assets to variables. These can be used to modify the appearance of the game and how the game is played. In general, here are types of objects that you can use in this language.

Variables

In general Orcinus variables can be classified into 2 groups, numeric and string

a = 312 // Numeric variable

b = "test" // String variable

Orcinus uses type inference, meaning that on declaration the type of the variable need not to be explicitly declared before the variable name. As a comparison, let's put the code of Java and Orcinus side by side to see the difference.

int a = 3 // Java's way

a = 3 // Orcinus' way
